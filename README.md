# FIXU AI Assets CDN

Public repository for hosting images and assets used in Notion, presentations, reports and other platforms.

## Structure

```
clientes/
  [cliente]/
    [contexto-fecha]/    <- images grouped by session/feedback round
```

## Usage

Raw URL format:
`https://raw.githubusercontent.com/isatabi03-afk/fixu-ai-assets/main/[path]`

- `cumbres/` — Cumbres Inmobiliaria (MIA)
